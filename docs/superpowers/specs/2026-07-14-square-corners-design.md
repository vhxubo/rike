# Rike 全局直角视觉调整

## 目标

移除页面和组件中的全部圆角设计，将纸本日课视觉统一为直角线框语言。

## 范围

- 移除所有 Tailwind `rounded-*` 类。
- 按钮、输入框、文本域、标签、分段控制器、导航、纸页、提示框、空状态和计划行全部使用直角。
- 开关轨道和滑块由胶囊形与圆形改为矩形。
- 统计条形图端点改为直角。
- 计划状态的空心圆和圆形勾选图标替换为方框和方框勾选图标。
- 黄色三角警报、日期、箭头和其他具有明确语义的图标保持原图形。

## 非目标

- 不改变颜色、间距、尺寸、边框宽度和布局。
- 不改变状态切换、模板、统计、持久化或日期滑动逻辑。
- 不重新设计图标库中与圆角无关的笔画端点。

## 实现方式

直接删除组件类名中的 `rounded-sm`、`rounded-md` 和 `rounded-full`，不新增全局覆盖规则。组件源码保持真实表达，避免未来出现局部圆角被全局 CSS 隐式覆盖。

状态图标使用 Lucide 的方形语义图标替换，按钮的可访问名称和交互行为保持不变。

## 验收

- `src` 中不存在 `rounded-*` 或 `border-radius`。
- 所有可见容器边角为直角。
- 开关和状态按钮不再包含圆形装饰。
- 原有测试文件继续保留。
- 本次属于普通视觉增量，不立即执行完整测试；在 `docs/testing/STATUS.md` 标记未执行状态。
