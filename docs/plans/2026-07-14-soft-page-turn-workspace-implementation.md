# Rike 柔软翻页与工作区增强实现计划

## 目标

依据 `docs/superpowers/specs/2026-07-14-soft-page-turn-month-view-design.md`，将日历改为日、周、月层级，重做柔软纸页翻动，并加入全局工具栏、统一返回、回到今天、本周总结、多范围总统计和三态主题。

## 实施顺序

### 1. 日历类型与 v3 迁移

- 将 `CalendarView` 的 `year` 替换为 `month`。
- 增加按月导航和固定 42 格月历日期生成工具。
- 将计划 store 升至 v3，v2 `year` 映射为 `month`，保留日期与记录。
- 更新缩放层级、范围导航、日期标题和备份契约类型。
- 测试日周月边界、跨年月导航和 v1/v2/v3 迁移。

### 2. 单月视图

- 用 `MonthView` 替换 `YearView`。
- 展示六周七列，补位日期使用弱化样式并保留状态。
- 点击任意日期原子进入对应日计划页。
- 更新视图切换和页面集成测试。

### 3. 柔软纸页运动

- 提取翻页视觉进度纯函数，计算平移、轻倾斜、目标页视差和光影强度。
- `PageTurn` 从 180°整页旋转改为横向移动和最高约 10°倾斜。
- 增加边缘高光、弯曲带和动态软阴影。
- 完成与回弹使用不同阻尼弹簧，按钮与拖动共用完成路径。
- 保留动画令牌、双指取消、交互区排除和减少动态效果降级。

### 4. 工作区导航与工具栏

- 建立 `calendar / week-summary / statistics` 会话页面模式和返回快照。
- 左上角统一渲染工具栏按钮或返回按钮。
- 工具栏提供本周总结、周视图、月视图和总统计入口。
- 日视图增加本周总结快捷按钮。
- 工具栏支持外部点击、Escape、焦点进入和焦点恢复。
- 日期标题区增加按页面语义工作的回到今天按钮。

### 5. 截至今天统计

- 建立通用日期范围和范围统计选择器。
- 统计计划总数、完成数、未完成数、完成率、计划文字、日结文字和六科未完成分布。
- 未来日期和计划全部排除；全部范围从不晚于今天的最早记录开始。
- 本周总结复用通用选择器。
- 总统计支持本周、本月、本年和全部，并实现范围导航。

### 6. 三态主题

- 建立独立主题偏好模块和持久化边界。
- 首次渲染前解析系统、日间或夜间主题并设置根节点属性。
- 系统模式监听 `prefers-color-scheme` 实时变化。
- 将现有颜色主题令牌改为可覆盖 CSS 变量，补充夜间纸本配色。
- 工具栏底部接入三段主题控件和实际生效说明。

### 7. 集成、文档与验证

- 覆盖工具栏入口、返回快照、回到今天、统计口径和主题切换测试。
- 更新需求记录、测试台账和交接文档。
- 执行类型检查、Lint、全部测试和生产构建。

## 风险控制

- 页面模式和返回快照不进入计划持久化，避免临时导航污染数据迁移。
- v3 迁移函数兼容 v1 缺少视图、v2 `year` 和 v3 非法值。
- 统计只通过纯函数读取记录，不写入汇总或修改计划。
- 月视图只渲染 42 个日期，不预渲染相邻月份页面之外的内容。
- 翻页视觉参数由纯函数控制，0 和 1 进度必须无残留阴影或倾斜。
- 主题偏好与计划 store 分离，存储失败不影响计划读写。
- 工具栏关闭和页面卸载必须恢复焦点并清理全局监听器。

- 手机和触控板上的柔软翻页跟手、完成与回弹。
- 双指缩放取消翻页及日、周、月层级切换。
- 工具栏浮层定位、键盘焦点、统一返回和回到今天。
- 日间、夜间和跟随系统时的首屏、纸面、状态色与翻页光影。
