# Rike 第三阶段实现计划

## 目标

依据 `docs/superpowers/specs/2026-07-14-calendar-views-gestures-design.md`，完成日、周、年视图，视图层级缩放，三种范围的 CSS 3D 跟手翻页，以及原生缩放、拖拽和非编辑文本选中的拦截。

## 实施顺序

### 1. 日期范围与概览派生

- 扩展 `src/features/plans/date.ts`：增加按周、按年导航，年份日期和月份网格生成，以及闰日跨年回退。
- 新建日概览纯选择器，统一派生空、未来、待处理、完成和未完成状态。
- 让周视图和年视图只消费选择器结果，不保存重复摘要。
- 覆盖空白周六、固定工作日、过去未决项、未来计划和闰年边界测试。

### 2. 日历游标与持久化迁移

- 将 `calendarView` 加入计划 store 持久化状态。
- 增加直接切换视图、按当前视图导航范围、从概览打开日视图的 action。
- 将持久化版本升为 v2，导出可测试的迁移函数。
- v1 迁移保留日期与全部记录并补充日视图；非法视图回退为日视图。

### 3. 公共日历工作区

- 把当前页面中的日期标题、每日一句、视图切换和内容区域重组为 `CalendarWorkspace`。
- 建立能按视图切换单位和标签的范围导航。
- 日视图复用现有 `WeekdayPlan`、`SaturdayPlan` 和 `SundaySummary`。
- 保持恢复中、恢复失败和减少动态效果的现有行为。

### 4. 周视图与年视图

- 实现七日周摘要：日期、总数、完成数、未完成数、整体状态和周日完成比例。
- 实现十二个月年网格：周一开头、月外留空、今天与选中日标记、计划状态标记。
- 周或年日期点击通过单个 store action 原子设置日期并进入日视图。
- 确保状态不仅依赖颜色，并提供明确按钮名称和当前状态文本。

### 5. 应用级原生行为拦截

- 更新 viewport，加入手机缩放限制。
- 建立应用启动时注册、卸载时清理的原生行为拦截器。
- 阻止 Safari gesture、多指 touchmove、Ctrl/Command wheel、缩放快捷键和 dragstart。
- 全局禁用非编辑文本选中，对输入、文本域和显式可编辑区域恢复文本选择。

### 6. 视图缩放手势

- 用纯函数实现视图层级、触点距离和方向判定。
- 用独立 hook 管理双指距离、触控板累计量、阈值、锁定和复位。
- 每次连续手势最多切换一级；双指开始时取消单指翻页。
- 分段控件保留为所有平台的稳定入口。

### 7. CSS 3D 跟手翻页

- 建立通用 `PageTurn` 容器，同时支持日、周、年范围。
- 拖动进度驱动旋转角度、背面亮度、边缘阴影和下一页显露。
- 用距离与速度决定完成或回弹；按钮触发自动翻页。
- 动画完成后才提交范围变更；翻动期间锁定目标页交互和重复导航。
- 输入、按钮、链接等交互区域不启动翻页；双指取得控制权时复位翻页。
- 减少动态效果时降级为即时切换和淡入。

### 8. 集成、文档与验证

- 更新页面测试，覆盖三视图切换、范围单位和点击日期进入日视图。
- 覆盖缩放阈值、单次锁定、翻页提交/回弹和原生事件拦截。
- 更新 `docs/testing/STATUS.md`、需求过程记录和第三阶段交接文档。
- 执行完整类型检查、Lint、测试和生产构建。

## 风险控制

- 手势判定保持为纯函数，DOM 监听器只负责采样和生命周期。
- 双指缩放优先于单指翻页，翻页意图成立前不阻止纵向滚动。
- 翻页只同时保留当前页和相邻目标页，年视图不预渲染多个年份。
- 持久化迁移独立测试，任何失败都不能覆盖 v1 记录。
- 全局非被动监听器必须成对清理，手势取消和窗口失焦必须复位锁定。
- 所有原生拦截都保留可编辑区域的输入、选择、复制和粘贴能力。
